# 🛠 Технические базисы (Technical Baselines)

## 📱 Мобильное приложение (Mobile Baseline)
**Цель:** Запуск в App Store / Google Play за 2-4 недели.

### 🧩 Обязательные модули:
- **Auth:** Firebase Auth / Supabase Auth.
- **Push Notifications:** Firebase Messaging.
- **Analytics:** Firebase / PostHog.
- **Error Tracking:** Sentry.

### 📦 UI Kit:
- Material 3 (Android) / Cupertino (iOS).
- Готовые компоненты для Login, Profile, Settings.

---

## 🌐 Веб-сервис / Landing (Web Baseline)
**Цель:** Высокий SEO и конверсия.

### 🧩 Обязательные модули:
- **SSR/SSG:** Next.js.
- **Forms:** React Hook Form + Zod.
- **Payments:** Stripe / Lemon Squeezy / CloudPayments.

### 📦 UI Kit:
- Shadcn UI / Tailwind UI.

---

## ⚙️ Backend / API (Service Baseline)
**Цель:** Масштабируемость и стабильность.

### 🧩 Обязательные модули:
- **ORM:** Prisma / SQLAlchemy.
- **Documentation:** Swagger (Autogenerated).
- **Security:** Helmet, CORS, Rate Limiting.
