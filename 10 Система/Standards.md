# 📏 Технические стандарты (Tech Standards)

## 📱 Мобильные приложения (Mobile)
- **Основной стек:** Flutter (Dart) или React Native (TypeScript).
- **Архитектура:** Clean Architecture / Feature-driven.
- **Состояние:** BLoC / Riverpod / Redux.
- **Обязательно:** Auth, Sentry (errors), Analytics, Dark Mode support.

## 🌐 Сайты и Сервисы (Web)
- **Фреймворк:** Next.js (TypeScript).
- **Стили:** Tailwind CSS.
- **Бэкенд:** Supabase (Auth + DB) или FastAPI.
- **Обязательно:** SEO, Core Web Vitals, Responsive Design.

## ⚙️ Сервисы и API (Backend)
- **Стек:** Python (FastAPI) или Node.js.
- **БД:** PostgreSQL (Relational) / Redis (Cache).
- **Документация:** Swagger / OpenAPI.
- **Обязательно:** CORS, JWT Auth, Rate Limiting.

---
*Для быстрого запуска MVP используем готовые UI Kit и Low-code интеграции там, где это оправдано.*
